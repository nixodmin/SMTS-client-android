# SMTS client android
Клиент для андроид-устройств для безопасного обмена текстовыми сообщениями и файлами (сквозное шифрование, сервер ничего не хранит, история не сохраняется).

Подключается к развёрнутому серверу https://github.com/nixodmin/SMTS

1) Сперва вы должны запустить серверный скрипт SMTS на Python или иметь следующие данные о таком сервере: его IP, порт и HANDSHAKE-пароль.
2) Установите клиент на андроид-устройство - либо соберите из исходников данного git-а, либо возьмите apk файл в Releases.
3) После установки клиента настройте подключение к серверу - правое верхнее меню "Подключение к серверу" (именно здесь надо вводить IP, порт и HANDSHAKE-пароль).
<img width="576" height="1280" alt="изображение" src="https://github.com/user-attachments/assets/e4048ce0-3ce0-4d29-912a-2b76c403e060" />

4) если подключение успешное в строке "Ваш текущий статус" вы получите свой ID: например я получил client_11.
5) Сообщите собеседнику свой ID или узнайте его ID и подключитесь.
<img width="576" height="1280" alt="изображение" src="https://github.com/user-attachments/assets/7c29fa96-a7b7-4b76-848d-15cf6b548ac0" />

6) Выберите собеседника (чекбокс). Можно выбрать несколько собеседников, тогда ваши сообщения или файлы будут отправлены одновременно нескольким контактам.
<img width="576" height="1280" alt="изображение" src="https://github.com/user-attachments/assets/dd7e9799-a1a5-425c-991a-cd3755edb3c6" />

7) Нажмите два раза на надпись "Ваш текущий статус", либо в верхнем правом меню выберите "Переключить панели", чтобы начать общение.
<img width="576" height="1280" alt="изображение" src="https://github.com/user-attachments/assets/aaa26097-2a57-4ac4-ab3d-34e37facf169" />

8) Можете безопасно обмениваться файлами до 10Мб (по умолчанию сохраняются в общую папку "Загрузки").

# Внимание

Важно: если вы используете энергосберегающий режим приложение будет постоянно отключаться от сервера.

Очень важно: не используйте публичные сервера SMTS если таковые появятся. Вы всегда можете развернуть свой сервер на домашнем ПК и пробросить его в интернет через любой reverse-proxy сервис (вроде localtonet).
